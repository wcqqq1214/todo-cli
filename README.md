# TaskMaster CLI

一个功能完整的命令行任务管理工具，使用 TypeScript 开发。

## 技术栈

- TypeScript
- Node.js
- Inquirer.js (交互式命令行)
- Chalk (终端颜色)
- cli-table3 (表格展示)
- date-fns (日期处理)

## 快速开始

### 安装依赖

```bash
pnpm install
```

### 开发模式

```bash
pnpm dev
```

### 构建

```bash
pnpm build
```

### 运行

```bash
pnpm start
```
